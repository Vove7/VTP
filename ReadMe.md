## Usage

* Step 1. Add the JitPack repository to your build file

gradle:
```groovy
allprojects {
    repositories {
        //...
        maven { url 'https://jitpack.io' }
    }
}
```

* Step 2. Add the dependency

gradle:
```groovy
dependencies {
    implementation 'com.github.Vove7:VTP:最新版本'
}
```
